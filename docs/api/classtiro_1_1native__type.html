◆ destroy()

void destroy ( const native &  object) const
inline

Manually destroys the native object.

Future access() calls to the same object will fail with an exception. Note that objects are destroyed automatically when they are collected by the garbage collector. This function allows the programmer to trigger the destruction at an earlier point in time.
