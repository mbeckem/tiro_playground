◆ view()

std::string_view view ( ) const
inline

Returns an unowned view into the string's storage without performing a copy.

The view is not zero terminated.

Warning
The view points into the string's current storage. Because objects may move on the heap (e.g. because of garbage collection) this data may be invalidated. The data may only be used immediately after calling this function, before any other vm function might allocate and therefore might trigger a garbage collection cycle.
